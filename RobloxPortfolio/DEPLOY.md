# Roblox Portfolio

This site is published using GitHub Pages.

## How to deploy

1. Push all files to a GitHub repository (e.g., `roblox-portfolio`).
2. In your repository, go to Settings > Pages.
3. Set the source branch to `main` (or `master`) and the folder to `/root` or `/`.
4. Save and wait for GitHub Pages to deploy your site.

Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

---

- Make sure your `index.html` is at the root of the repository.
- For custom domains, configure in the GitHub Pages settings.
